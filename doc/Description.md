# App name

Spop

# Short description

```
Einkaufsliste für Wochenendshopper
```

# Full description

```
Spop Ist eine App um wöchentliche Haushaltseinkäufe erträglicher zu machen. 

Schon wieder so eine Shopping App?

Ja, richtig, noch eine, aber... Bei der hier dreht es sich alles um die "Dinge", die eingekauft werden sollen und nicht um "Listen". Und das möglichst schnell, einfach, unkompliziert und immer wieder. 

Oookay, toll, und was soll das heißen?

Ein Ding hängt nicht in einer Liste fest. Es muss nicht kopiert, verschoben, gelöscht werden, sondern wird einfach im Vorratsort oder Shop an- oder abgehakt. 

Es gibt zwei wesentliche Sichten in Spop:

	1. Erstellen - Vorrat
	2. Einkaufen - Shops

Erstellen - Vorrat:

Zumindest ich gehe vor dem Wocheneinkauf durch die Wohnung von Vorratsort zu Vorratsort um zu schauen was fehlt. Also, z.B. Kühlschrank auf, schauen was fehlt, Kühlschrank zu, Wiederholen. Nächster Vorratsort.
In Spop gibt es daher für jeden erstellten Vorratsort eine Historie.

Einkaufen - Shops

Meistens fahre ich beim Einkaufen die selben Shops in der selben Reihenfolge ab. Vom günstigsten zu den weniger günstigen. Was ich in dem einen nicht bekomme, kaufe ich dann im nächsten. Also gibt es in Spop mehrere Shops. Die nicht abgehakten Dinge werden in den Folgenden angezeigt - bis zu einem Stopper.
Jeder Shop hat eine eigene Reihenfolge der Kategorien um die Sortierung der Dinge im Shop festlegen zu können. 
Pro Shop gibt es einen Preis, den man eingeben kann. Der günstigste wird hervor gehoben. 

Ach ja: ohne Kosten, ohne Werbung, ohne Datensammeln. Im Moment alles Offline, keine Synchronisierung von Daten, nur ein Shopper.
```
