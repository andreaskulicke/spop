# App name

```
Spop
```

# Short description

```
Einkaufsliste für Wochenendshopper
```

# Full description

```
Spop Ist eine App um wöchentliche Haushaltseinkäufe erträglicher zu machen. 

Schon wieder so eine Shopping App?

Ja, richtig, noch eine, aber... Bei der hier dreht es sich alles um die "Dinge". Dinge, die eingekauft werden sollen und nicht um "Listen". Und das möglichst schnell, einfach, unkompliziert und immer wieder. 

Oookay, toll, und was soll das heißen?

Ein Ding hängt nicht in einer Liste fest. Es muss nicht aus Listen kopiert, verschoben, gelöscht werden, sondern wird einfach im Vorratsort oder Shop an- oder abgehakt. 

Es gibt zwei wesentliche Sichten in Spop:

	1. Erstellen - Vorratsorte
	2. Einkaufen - Shops

Erstellen - Vorratsorte:

Vor dem Wocheneinkauf gehe ich durch die Wohnung von Vorratsort zu Vorratsort um zu schauen was fehlt. Also, z.B. Kühlschrank auf, schauen was fehlt, Kühlschrank zu, Wiederholen. Nächster Vorratsort.
In Spop gibt es für jeden erstellten Vorratsort eine eigene Historie. Damit lassen sich wiederkehrende Einkäufe schnell erstellen. 

Einkaufen - Shops

Meistens fahre ich beim Einkaufen die selben Shops in der selben Reihenfolge ab. Vom Günstigen zu den weniger Günstigen (Aldi - Edeka). Oder vom Spezielleren zum Allgemeinen (Obstladen - Supermarkt). Was ich in dem einen nicht bekomme, kaufe ich dann im nächsten. Dafür gibt es in Spop mehrere Shops. Die nicht abgehakten Dinge im früheren Shop werden in den Folgenden angezeigt - bis zu einem Stopper.
Jeder Shop hat eine eigene Reihenfolge der Kategorien, um die Sortierung der Dinge im Shop festlegen zu können. 
Pro Shop gibt es einen Preis, den man eingeben kann. Der Günstigste wird hervor gehoben. 

Weitere Features:

- Taschenrechner für schnellere Eingabe
- Themes: hell, dunkel und entsprechende Farben
- Drag and drop von: Vorratsorten, Shops, Kategorien
- Standard Kategorie für Vorratsorte, Shops
- ...

Ach ja: ohne Kosten, ohne Werbung, ohne Datensammeln. Im Moment alles Offline, keine Synchronisierung von Daten - nur ein Shopper.
```
